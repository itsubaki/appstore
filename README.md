# apst

 - app store command line tool.

## Install

```
$ go get github.com/itsubaki/apst
$ go install github.com/itsubaki/apst
```

## Usage

```
$ apst top gungho
4: パズル＆ドラゴンズ(jp.gungho.pad) [GungHo Online Entertainment, Inc.]
59: ディズニー マジックキングダムズ(Disney Magic Kingdoms)(jp.gungho.dmk) [GungHo Online Entertainment, Inc.]
128: ケリ姫スイーツ(jp.gungho.kerihimequest2) [GungHo Online Entertainment, Inc.]
160: サモンズボード(jp.gungho.bm) [GungHo Online Entertainment, Inc.]
```

```
$ apst top gungho --limit 100
4: パズル＆ドラゴンズ(jp.gungho.pad) [GungHo Online Entertainment, Inc.]
59: ディズニー マジックキングダムズ(Disney Magic Kingdoms)(jp.gungho.dmk) [GungHo Online Entertainment, Inc.]
```
